🤖 Model & Training

  • Model: DistilBERT
  • Tokenizer: DistilBertTokenizerFast
  • Data Split: 80% Train / 20% Test
  • Epochs: 3
  • Learning Rate: 2e-5
  • Batch Size: 16

ℹ️ Project Info

This project fine-tunes a transformer model to classify Reddit posts into five mental health categories. The goal is to create a reliable tool for identifying different mental health discussions automatically.

Data Source: Kaggle
